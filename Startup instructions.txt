1 In the project root, run pip install -r backend/requirements.txt to install Flask. 

STARTUP INSTRUCTIONS                                                                    
 2 Run python backend/app.py to start the backend on http://localhost:5000.                                                                               
 3 In a separate terminal, run npm install inside the frontend/ folder, then npm run dev to start the React dev server on http://localhost:3000.          
 4 Open http://localhost:3000 in your browser. You should see "Backend Status: Backend is running".
